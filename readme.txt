# Professional Portfolio Website

This project is a responsive professional portfolio website created using HTML5 and CSS3. It showcases skills, projects, certifications, and contact information.

## HTML Tags Used

1. <!DOCTYPE html>: Declares the document type as HTML5
2. <html>: Root element of the HTML page
3. <head>: Contains meta information about the document
4. <meta>: Provides metadata about the HTML document
5. <title>: Specifies a title for the HTML page
6. <link>: Links to external resources (CSS file and favicon)
7. <body>: Defines the document's body
8. <header>: Represents a container for introductory content
9. <h1>, <h2>: Heading elements
10. <nav>: Defines a set of navigation links
11. <menu>: Represents a list of commands
12. <li>: Defines a list item
13. <a>: Defines a hyperlink
14. <main>: Specifies the main content of a document
15. <section>: Defines a section in a document
16. <img>: Embeds an image
17. <p>: Defines a paragraph
18. <ul>: Defines an unordered list
19. <table>: Defines a table
20. <thead>: Groups the header content in a table
21. <tbody>: Groups the body content in a table
22. <tr>: Defines a row in a table
23. <th>: Defines a header cell in a table
24. <td>: Defines a cell in a table
25. <div>: Defines a section in a document
26. <figure>: Specifies self-contained content
27. <figcaption>: Defines a caption for a <figure> element
28. <form>: Defines an HTML form for user input
29. <label>: Defines a label for an <input> element
30. <input>: Defines an input control
31. <textarea>: Defines a multiline input control
32. <button>: Defines a clickable button
33. <footer>: Defines a footer for a document or section
34. <audio>: Embeds sound content
35. <video>: Embeds video content
36. <source>: Specifies multiple media resources for <audio> and <video>
37. <details>: Defines additional details that the user can view or hide
38. <summary>: Defines a visible heading for a <details> element

## CSS Properties and Techniques Used

1. Box-sizing
2. Flexbox
3. Float
4. Overflow
5. Position (relative and absolute)
6. Media queries for responsiveness
7. Transitions for hover effects
8. CSS Grid for form layout

This project demonstrates the use of modern HTML5 elements and CSS3 properties to create a responsive and visually appealing portfolio website.
